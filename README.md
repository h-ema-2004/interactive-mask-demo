# Interactive 3D Mask Demo

صفحة تجريبية تفاعلية لمنتج LED therapy mask باستخدام:

- Three.js / WebGL
- GSAP ScrollTrigger
- HTML + CSS + JavaScript في ملف واحد
- GitHub Pages للنشر

## رابط الموقع

بعد نجاح GitHub Pages، يفتح الموقع من:

https://h-ema-2004.github.io/interactive-mask-demo/

## ملاحظات

- الصفحة تحتاج إنترنت لأنها تستورد Three.js و GSAP من CDN.
- المجسم الحالي Demo برمجي داخل Three.js وليس ملف GLB حقيقي.
- يمكن لاحقًا استبدال المجسم بملف `mask.glb` حقيقي بعد تحسينه في Blender و glTF Transform.
